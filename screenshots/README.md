# Screenshots Folder

## Instructions

Place your Git installation verification screenshot in this folder.

### File Naming Convention

Name your screenshot file using the following format:
```
git_installation_<your_name>.<extension>
```

**Examples:**
- `git_installation_john_doe.png`
- `git_installation_jane_smith.jpg`
- `git_installation_alex_johnson.png`

### Acceptable File Formats

- PNG (`.png`) - Recommended
- JPEG (`.jpg` or `.jpeg`)
- GIF (`.gif`)

### Screenshot Requirements

Your screenshot MUST show:
1. ✅ Git Bash (Windows) or Terminal (macOS/Linux) window
2. ✅ The `whoami` command you typed
3. ✅ The output showing your username
4. ✅ Clear and readable text

### Adding Your Screenshot

1. Take your screenshot following the instructions in the main README.md
2. Save it with the correct naming convention
3. Copy or move the file to this `screenshots/` folder
4. Commit and push your changes:
   ```bash
   git add screenshots/git_installation_<your_name>.png
   git commit -m "Add Git installation verification screenshot"
   git push
   ```

---

**Note:** Do not modify or delete the `.gitkeep` file in this folder.
